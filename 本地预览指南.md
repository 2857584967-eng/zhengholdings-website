# 本地预览指南

## 立即查看网站（无需部署）

### 方法1：直接打开（最简单）
1. 进入文件夹：`zhengholdings-website/`
2. 双击 `index.html` 文件
3. 浏览器会自动打开网站

### 方法2：使用本地服务器（推荐）
```bash
# 方法A：Python（Windows/Mac/Linux都可用）
cd zhengholdings-website
python -m http.server 8000

# 然后在浏览器访问：http://localhost:8000

# 方法B：Node.js（如果有安装）
cd zhengholdings-website
npx serve .

# 然后在浏览器访问：http://localhost:3000
```

### 方法3：使用VSCode Live Server扩展
1. 安装VSCode的"Live Server"扩展
2. 右键点击 `index.html` → "Open with Live Server"
3. 自动在浏览器打开：http://localhost:5500

## 部署到Netlify的快速步骤

### 准备工作
1. 访问：https://app.netlify.com
2. 使用GitHub/GitLab/Google/邮箱注册

### 部署步骤
1. 点击 **"Add new site"** → **"Import an existing project"**
2. 选择 **Git** → **Gitee**
3. 授权访问您的Gitee账号
4. 选择仓库：`guangdong-zhenghe-holdings_0/zhengholdings-website`
5. 点击 **"Deploy site"**
6. 等待1-2分钟，获取您的专属网址

### 预计网址格式
- `https://zhengholdings-website.netlify.app`
- 或 `https://随机名称.netlify.app`
