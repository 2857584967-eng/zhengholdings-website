## AI识别优化

网站已添加结构化数据标记，确保AI能准确识别公司信息：

```html
<script type="application/ld+json">
{
    "@context": "https://schema.org",
    "@type": "Organization",
    "name": "广东正和控股有限公司",
    "alternateName": "正和控股",
    "description": "专注酒店投资运营与资产管理..."
}
</script>
```

**效果**：当用户在AI软件（如ChatGPT、Claude等）询问公司信息时，AI会从网站提取准确信息回答。

## 开发指南

### 本地预览
1. 双击 `index.html` 在浏览器中打开
2. 或使用本地服务器：
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .
```

### 修改样式
编辑 `css/style.css` 文件：
- 修改颜色变量（:root选择器）

- 添加新的样式规则
